## What is ElarisLabs?

ElarisLabs is an AI-native creative operating system for brands.

You onboard from a URL, build on an infinite canvas, generate at scale, edit
video, publish, and learn. It all runs in one continuous loop.

These docs cover how to connect ElarisLabs to your own tools and agents. Once
connected, you can drive **Creative Studio** from wherever you work. That
includes images, video, audio, brand packs, and composer edits.

## How it fits together

Your prompt travels from an MCP client, such as Claude or Cursor, to the hosted
ElarisLabs MCP endpoint. The endpoint runs Creative Studio tools and bills
credits to the brand your API key is bound to.

<Steps>
  <Step title="Create a key">
    Generate a brand-scoped API key in your profile.
  </Step>
  <Step title="Connect a client">
    Paste a small JSON config into Claude, Cursor, or any MCP client.
  </Step>
  <Step title="Generate">
    Ask in plain language. The agent discovers tools and runs them for you.
  </Step>
</Steps>
