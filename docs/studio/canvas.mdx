The Creative Studio canvas is an infinite workspace where you arrange and connect AI nodes. Pan and zoom freely to organize complex pipelines. Select any node to edit its parameters in the contextual panel on the right.

## How do you navigate the canvas?

| Action | How to do it |
|---|---|
| **Pan** | Click and drag on empty canvas space |
| **Zoom** | Scroll wheel or pinch gesture |
| **Select a node** | Click the node header |
| **Multi-select** | Click and drag a selection box over multiple nodes |
| **Move nodes** | Drag selected nodes to reposition |
| **Connect nodes** | Drag from an output port to an input port |

## How do you build a node workflow?

<Steps>
  <Step title="Add a node">
    Click the **+** button in the toolbar or right-click the canvas and choose a node type.
  </Step>
  <Step title="Choose mode and model">
    Select the operation mode (e.g. text-to-image vs. remove background). Choose a specific model or leave it on `auto` to let the router select a strong default for your brief.
  </Step>
  <Step title="Set parameters">
    Fill in your prompt, aspect ratio, resolution, and any other mode-specific inputs in the contextual panel.
  </Step>
  <Step title="Connect upstream outputs">
    Drag a connection from an upstream node's output port to feed reference images, source clips, or audio into the current node.
  </Step>
  <Step title="Generate">
    Click **Run**. Results render inline on the node and become available as outputs for downstream steps.
  </Step>
</Steps>

<Tip>
Use `auto` for image model selection when you want the router to pick a strong default for your brief — it routes to the best available model based on prompt complexity and style.
</Tip>

## How do projects work?

Projects save your entire canvas state — the node graph, model selections, parameter settings, and media references.

<AccordionGroup>
  <Accordion title="Save a project">
    Click **Save** in the top bar. Your canvas is persisted under your brand's Studio project library with the current name.
  </Accordion>
  <Accordion title="Load a project">
    Open the project picker from the top bar or brand dashboard and select any previously saved project. The full graph and media references are restored.
  </Accordion>
  <Accordion title="Rename a project">
    Click the project name in the top bar to rename inline. Descriptive names make projects discoverable in the asset library and easier to share.
  </Accordion>
  <Accordion title="Share a project">
    Generate a **public view link** (read-only) or a **team share token** for collaborators inside your organization. See [Collaboration](/studio/collaboration) for details.
  </Accordion>
</AccordionGroup>
