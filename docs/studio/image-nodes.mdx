Image nodes are the foundation of Creative Studio's visual pipeline. Each node runs one image operation: pick a **mode**, choose a **model** (or leave it on **Auto**), set parameters, and generate. Chain nodes to build multi-step editing flows.

## Auto Router & the model picker

Every image node has a model selector. Leave it on **Auto** to let the router choose the best model for your prompt and mode, or open the picker to choose one yourself — each option shows its speed and credit cost.

<Tip>
Resolution and quality change credit cost. Higher resolutions (2K / 4K) and premium models cost more per run — check the estimate in the node panel before generating.
</Tip>

## What generation modes are available?

Create new images from text or from existing references.

### Text to image

Generate a new image from a text prompt. Set aspect ratio, resolution, and optional style references. Works with Auto Router and all major models.

**Best for:** hero shots, concept art, product visuals, ad backgrounds.

### Image to image

Condition generation on one or more reference images. Control how closely the output follows the reference via a strength / influence slider.

**Best for:** style transfer, iterating on existing shots, brand-consistent generation.

## Which models are available?

Open the model picker on any image node for featured options and full provider lists. Common families include:

| Model / Family | Strengths |
|---|---|
| **Flux** | High-quality photorealism and prompt adherence |
| **Nano Banana (Gemini)** | Fast generation, strong at text-in-image |
| **Seedream** | Stylized and artistic outputs |
| **GPT Image** | Instruction-following, multi-subject scenes |
| **Kling** | Cinematic quality, strong composition |
| **Wan** | Versatile general-purpose generation |
| **Runway** | Creative / editorial style |
| **Ideogram utilities** | Typography and design |
| **Phota** | Portrait and product photography style |
| **Grok Imagine** | Expressive and imaginative outputs |
| **Hunyuan** | Asian aesthetic, high detail |
| **Quiver Arrow** | SVG vector generation |

Full matrix: [Image models](/models/image).
