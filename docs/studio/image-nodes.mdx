Image nodes are the foundation of Creative Studio's visual pipeline. Each node runs one image operation. Pick a **mode**, choose a **model** (or use `auto`), set your parameters, and generate. Chain nodes together to build multi-step editing flows.

## What generation modes are available?

<Tabs>
  <Tab title="Text to image">
    Generate a new image from a text prompt. Set aspect ratio, resolution, and optional style references. Works with all major models.

    **Best for:** hero shots, concept art, product visuals, ad backgrounds.
  </Tab>
  <Tab title="Image to image">
    Condition generation on one or more reference images. Control how closely the output follows the reference via a strength/influence slider.

    **Best for:** style transfer, iterating on existing shots, brand-consistent generation.
  </Tab>
</Tabs>

## What editing modes are available?

<Tabs>
  <Tab title="Upscale">
    Increase resolution and add fine detail to an existing image. Useful as a final step before export or handoff.

    **Input:** any image node output or uploaded asset.
  </Tab>
  <Tab title="Reframe">
    Outpaint an image to a new aspect ratio — expand the canvas on any edge. Ideal for adapting a portrait crop to landscape without re-generating.

    **Input:** any image.
  </Tab>
  <Tab title="Mask / inpaint">
    Draw a mask over a region and describe what should replace it. The rest of the image stays untouched.

    **Input:** source image + mask drawing.
  </Tab>
  <Tab title="Erase">
    Remove an unwanted object or region and fill it with the surrounding background naturally.

    **Input:** source image + selection.
  </Tab>
  <Tab title="Replace background">
    Automatically segment the subject and swap the background scene with a new one described by prompt.

    **Input:** source image + prompt for the new background.
  </Tab>
  <Tab title="Remove background">
    Cut out the subject and export with a transparent background (PNG alpha). Commonly used before the layer editor or brand pack.

    **Input:** any image.
  </Tab>
</Tabs>

## What SVG modes are available?

<CardGroup cols={2}>
  <Card title="Text to SVG" icon="pen-tool">
    Generate scalable vector artwork directly from a text description using Quiver Arrow.
  </Card>
  <Card title="Image to SVG" icon="scan-line">
    Vectorize any raster image into clean SVG paths — useful for logos, icons, and print-ready artwork.
  </Card>
</CardGroup>

## Which models are available for image nodes?

Creative Studio includes models from the following providers. Select from the model dropdown on any image node, or use `auto` to let the router choose.

| Model / Family | Strengths |
|---|---|
| **Flux** | High-quality photorealism and prompt adherence |
| **Nano Banana (Gemini)** | Fast generation, strong at text-in-image |
| **Seedream** | Stylized and artistic outputs |
| **GPT Image** | Instruction-following, multi-subject scenes |
| **Kling** | Cinematic quality, strong composition |
| **Wan** | Versatile general-purpose generation |
| **Runway** | Creative / editorial style |
| **Ideogram utilities** | Typography and design |
| **Phota** | Portrait and product photography style |
| **Grok Imagine** | Expressive and imaginative outputs |
| **Hunyuan** | Asian aesthetic, high detail |
| **Quiver Arrow** | SVG vector generation |

See [Image models](/models/image) for a full capability matrix including resolution support and credit costs.

<Note>
Resolution and quality settings directly affect credit cost. Higher resolutions (2K / 4K) and premium models cost more per generation. Check the credit estimate in the node panel before running.
</Note>
